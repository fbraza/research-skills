# Evaluation Assets — functional-enrichment-from-degs

This directory contains test cases and benchmarking materials for the
`functional-enrichment-from-degs` workflow.

## Contents

| File/Directory | Description |
|----------------|-------------|
| `simple_test.py` | Mock + live tests for ORA/GSEA logic (see below) |
| `test_results/` | Pre-computed test outputs (generated by running the workflow) |

## Running Tests

### Quick mock test (~5 seconds, no dependencies)
```bash
python assets/eval/simple_test.py
```

### Full live test (requires R + clusterProfiler)
```bash
python assets/eval/simple_test.py --live --verbose
```

## Test Coverage

| Test | What it checks |
|------|---------------|
| Gene list preparation | Correct filtering by padj ≤ 0.05 and \|log2FC\| ≥ 1 |
| ORA logic | Fisher's exact test + BH correction on mock gene sets |
| GSEA ranked list | Correct sorting, no NaN/Inf, correct gene ordering |
| Output file structure | CSV outputs can be created with expected columns |
| Live workflow (optional) | Full R workflow with airway dataset DE results |

## Expected Test Results (airway dataset)

When running the live test with the airway dexamethasone dataset:
- **Total genes:** ~15,000–20,000
- **Significant DEGs (padj ≤ 0.05, |log2FC| ≥ 1):** ~300–600
- **Top enriched pathways (upregulated):** Glucocorticoid receptor signaling, steroid response
- **Top enriched pathways (downregulated):** Inflammatory response, cytokine signaling, NF-κB

## test_results/ Directory

This directory stores pre-computed outputs from a reference run of the workflow.
It is populated by running the full workflow on the airway example dataset:

```r
source("scripts/load_example_data.R")
de_results <- load_airway_de_results()
# ... run full workflow ...
# Outputs saved to assets/eval/test_results/
```

**Note:** Binary files (`.rds`, `.pkl`) and plots are not tracked in version control.
Only CSV and markdown outputs are stored here.
