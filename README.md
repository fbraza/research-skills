# Academic research skills

This repository contains skills used for bioinformatics and agent-assisted analysis workflows.

It is structured as a Pi package and includes reusable resources for [Pi](https://pi.dev).

## Skills

* [`/ananse-grn`](./ananse-grn) - Infer gene regulatory programs with ANANSE-focused workflows.
* [`/bulk-omics-clustering`](./bulk-omics-clustering) - Cluster bulk omics samples with normalization-first best practices.
* [`/bulk-rnaseq-counts-to-de-deseq2`](./bulk-rnaseq-counts-to-de-deseq2) - Run bulk RNA-seq differential expression from counts with DESeq2.
* [`/cell-cell-communication`](./cell-cell-communication) - Analyze ligand-receptor signaling and cell-cell communication patterns.
* [`/cellbender`](./cellbender) - Clean droplet-based single-cell data with CellBender workflows.
* [`/chip-atlas-diff-analysis`](./chip-atlas-diff-analysis) - Use ChIP-Atlas for differential binding and enrichment-style analyses.
* [`/chip-atlas-peak-enrichment`](./chip-atlas-peak-enrichment) - Test genomic regions for ChIP-Atlas peak enrichment.
* [`/chip-atlas-target-genes`](./chip-atlas-target-genes) - Retrieve and prioritize target genes from ChIP-Atlas evidence.
* [`/clinicaltrials-landscape`](./clinicaltrials-landscape) - Map the clinical trial landscape for a disease, target, or modality.
* [`/coexpression-network`](./coexpression-network) - Build and interpret co-expression networks such as WGCNA-style analyses.
* [`/decode-deconvolution`](./decode-deconvolution) - Perform DECODE-based cell type deconvolution workflows.
* [`/disease-progression-longitudinal`](./disease-progression-longitudinal) - Model longitudinal disease progression and repeated-measures trajectories.
* [`/experimental-design-statistics`](./experimental-design-statistics) - Review study design, statistical assumptions, and analysis readiness.
* [`/functional-enrichment-from-degs`](./functional-enrichment-from-degs) - Run ORA/GSEA-style functional enrichment from DEG results.
* [`/genetic-variant-annotation`](./genetic-variant-annotation) - Annotate and prioritize genetic variants with external knowledge bases.
* [`/grn-pyscenic`](./grn-pyscenic) - Infer regulons and gene regulatory networks with pySCENIC.
* [`/gwas-to-function-twas`](./gwas-to-function-twas) - Connect GWAS signals to genes and function via TWAS-style workflows.
* [`/knowhows`](./knowhows) - Reusable methodological guidance and best-practice know-how references.
* [`/lasso-biomarker-panel`](./lasso-biomarker-panel) - Discover compact biomarker panels with LASSO/elastic-net style modeling.
* [`/literature-preclinical`](./literature-preclinical) - Search and summarize preclinical literature for translational questions.
* [`/literature-review`](./literature-review) - Gather and synthesize citation-backed scientific literature.
* [`/mendelian-randomization-twosamplemr`](./mendelian-randomization-twosamplemr) - Run two-sample Mendelian randomization analyses.
* [`/multi-omics-integration`](./multi-omics-integration) - Integrate multiple omics layers with method-selection guidance.
* [`/pcr-primer-design`](./pcr-primer-design) - Design and assess PCR primers with quality-control checks.
* [`/polygenic-risk-score-prs-catalog`](./polygenic-risk-score-prs-catalog) - Work with PRS Catalog-derived polygenic risk score workflows.
* [`/pooled-crispr-screens`](./pooled-crispr-screens) - Analyze pooled CRISPR screening data and hits.
* [`/proteomics-diff-exp`](./proteomics-diff-exp) - Perform differential protein expression analysis for proteomics datasets.
* [`/scaden-deconvolution`](./scaden-deconvolution) - Run Scaden-based transcriptomic deconvolution.
* [`/scientific-audit`](./scientific-audit) - Audit analysis steps for methodological and reporting correctness.
* [`/scientific-visualization`](./scientific-visualization) - Create publication-oriented scientific figures and visual outputs.
* [`/scientific-writing`](./scientific-writing) - Draft and revise papers, grants, rebuttals, and scientific prose.
* [`/scrna-trajectory-inference`](./scrna-trajectory-inference) - Infer single-cell trajectories, pseudotime, and lineage structure.
* [`/scrnaseq-scanpy-core-analysis`](./scrnaseq-scanpy-core-analysis) - Run core single-cell RNA-seq analysis in Python with Scanpy.
* [`/scrnaseq-seurat-core-analysis`](./scrnaseq-seurat-core-analysis) - Run core single-cell RNA-seq analysis in R with Seurat.
* [`/scvi-tools-scrna`](./scvi-tools-scrna) - Use scvi-tools models for advanced single-cell RNA-seq workflows.
* [`/scvi-tools-spatial`](./scvi-tools-spatial) - Apply scvi-tools methods to spatial transcriptomics problems.
* [`/spatial-transcriptomics`](./spatial-transcriptomics) - Analyze spatial transcriptomics data end to end.
* [`/survival-analysis-clinical`](./survival-analysis-clinical) - Perform survival and clinical outcome analyses.
* [`/upstream-regulator-analysis`](./upstream-regulator-analysis) - Identify candidate upstream regulators driving observed signatures.
